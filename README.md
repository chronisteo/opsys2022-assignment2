## CPU Scheduling Algorithms (C)

This project was developed for the **Operating Systems course** and implements basic **CPU scheduling algorithms** in C. The goal is to simulate how an operating system schedules processes for CPU execution.

The implemented scheduling algorithms include:

* First Come First Served (FCFS)
* Shortest Job First (SJF)
* Shortest Remaining Time First (SRTF)
* Round Robin (RR)

Each algorithm schedules processes based on their **arrival time** and **CPU burst time**, demonstrating the differences between preemptive and non-preemptive scheduling strategies.
